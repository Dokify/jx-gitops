## jx-gitops namespace

Updates all kubernetes resources in the given directory to the given namespace

***Aliases**: ns*

### Usage

```
jx-gitops namespace
```

### Synopsis

Updates all kubernetes resources in the given directory to the given namespace

### Examples

  # updates the namespace of all the yaml resources in the given directory
  jx-gitops namespace -n cheese --dir .

### Options

